# 自定义识别词助手

MoviePilot插件，用于可视化管理自定义识别词，支持标签分类和智能识别。

## 使用说明


### 分类管理

#### 创建分类
1. 点击"添加分类"按钮
2. 输入分类名称
3. 确认创建

#### 创建子分类
1. 在分类行点击"文件夹+"图标
2. 输入子分类名称
3. 确认创建

### 添加识别词

#### 方式一：快速添加
1. 点击"快速添加"按钮
2. 选择分类（可选，留空自动分配到"未分类"）
3. 选择识别词类型
4. 填写相应字段
5. 点击"添加"

#### 方式二：直接输入原始数据
```
# 动漫分类
测试屏蔽词
原词 => 替换词
前定位 <> 后定位 >> EP+1

# 电影分类
电影屏蔽词
电影原词 => 电影替换词
```

### 注意事项

1. 修改识别词前建议备份原始数据
2. 确保识别词格式符合要求
3. 修改后记得点击"保存修改"按钮
